# ix flake quickstart

- Enable direnv: `direnv allow .` (uses `.envrc` to enter the flake shell).
- Run ix from the flake: `nix run .#ix -- --help`.
- Dev shell with toolchain: `nix develop` (adds curl/wget/git plus clang/llvm/cmake/ninja and ix itself).
- Update the pinned upstream:
  1. Set `ixRev`, `ixSha256`, and `ixVersion` in `flake.nix`.
  2. Refresh the lockfile: `nix flake lock --update-input nixpkgs`.
- Runtime tools are provided via the wrapper PATH; extend `runtimeTools` or `buildTools` in `flake.nix` if ix recipes demand more host utilities.
